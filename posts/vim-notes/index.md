## widows {#widows}

_**C-w s**_
: horizontal split window

_**C-w v**_
: vertical split window

_**C-w q**_
: quit a window

_**C-w h/j/k/l**_
: move between windows

_**C-w H/J/K/L**_
: make current master window

_**C-w +/-**_
: inc/dec height, can prepend with a number

_**C-w >/<**_
: inc/dec width, can prepend with a number


## jump {#jump}

_**C-o**_
: jump back

_**C-i**_
: jump forward
